# 📇 Card Stack Manager - Interactive Call Card System

A beautiful, interactive web application for managing call cards with local storage persistence. Create, organize, filter, and manage contact cards with a smooth 3D stacking effect.

![Card Stack Manager Demo](https://call-card-localstorage.netlify.app//)

## ✨ Features

- **📝 Create Cards** - Add new cards with image, name, hometown, purpose, and category
- **💾 Local Storage** - All cards are automatically saved in your browser
- **🎨 3D Stack Effect** - Beautiful card stacking animation with depth layers
- **🔍 Filter System** - Filter cards by categories (Emergency, Important, Urgent, No Rush)
- **⬆️⬇️ Navigation** - Easy up/down navigation through cards
- **🗑️ Delete Cards** - Right-click to delete unwanted cards
- **📱 Responsive Design** - Works perfectly on mobile, tablet, and desktop
- **⚡ Smooth Animations** - Fluid transitions and hover effects
